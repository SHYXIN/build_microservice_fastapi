=========================== 基本信息 ==================================
应用名称: 在线学术讨论论坛
状态: 原型阶段
完成日期: 2021年7月30日

=========================== 描述 ====================================
此应用程序旨在为学生、教师、家庭成员或组织中的用户提供在线知识构建和交流机制支持。
这是一个微服务应用程序，能够在用户之间建立思想交流并讨论无限的话题，包括社会活动和教育理念。

此应用程序使用 FastAPI 框架，由 Python 3.8 驱动。
目前尚未包含数据库。

requirements.txt 文件将指引您安装所需依赖项。只需运行以下命令安装:
```bash
pip install -r requirements.txt
```
